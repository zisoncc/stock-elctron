股票基金行情显示软件 - 测试页面

功能特点

  • 自定义股票列表，支持添加和删除股票
  • 窗口支持拖拽移动，保存位置状态
  • 鼠标悬停显示详细行情数据
  • 右键菜单支持删除股票
  • 自动数据自动刷新（30秒）
  • 支持最小化窗口
  • 详细行情包含价格走势图

使用说明

1. 打开主程序：启动股票行情显示

2. 拖拽窗口：点击标题栏顶部标题栏拖动窗口

3. 查看详情：鼠标悬停在股票项上

4. 删除股票：右键击股票项上右键点击"删除"

5. 刷新数据：点击顶部的刷新按钮

6. 最小化：点击顶部的减号按钮

添加股票

在浏览器控制台中执行以下代码添加股票：

addStock('sh600000'); // 添加浦发银行
addStock('sz000001'); // 添加平安银行
addStock('sh600519'); // 添加贵州茅台

股票代码格式：沪市股票以"sh"开头，深市股票以"sz"开头

技术实现

本软件使用以下技术实现：

  • HTML5 + CSS3 + JavaScript
  • Tailwind CSS 用于样式
  • Chart.js 用于价格图表
  • 腾讯股票API 用于行情数据
